# Tools & Software to Practice With

## DTN Software (Free)

| Tool | Language | Link | Description |
|------|----------|------|-------------|
| **ION-DTN** | C | [SourceForge](https://sourceforge.net/projects/ion-dtn/) | JPL's reference BPv7 implementation. Install and experiment with bundle creation. |
| **DTN7** | Go | [GitHub](https://github.com/dtn7/dtn7) | Modern BPv7 implementation. Easier to set up than ION. |
| **IBR-DTN** | C++ | [GitHub](https://github.com/ibrdtn/ibrdtn) | Academic DTN implementation |
| **pyDTN** | Python | [GitHub](https://github.com/nicktehrany/pyDTN) | Python-based DTN, good for learning |
