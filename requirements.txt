# AETHERIX — Python Dependencies
# ============================================================
#
# CORE SIMULATION (src/) + ENTRY POINT (run_simulation.py):
#   ZERO external dependencies. Every module (link budgets, RL routing,
#   QKD, orbital mechanics, radiation hardening, the DTN simulator) runs
#   on the Python standard library alone. This is a deliberate design
#   choice — it makes the architecture trivially reproducible:
#
#       git clone https://github.com/matx104/AETHERIX.git
#       cd AETHERIX
#       python run_simulation.py          # runs with no extra installs
#
# The production roadmap (DQN, JPL Horizons ephemeris, ns-3) is documented
# in README.md; those heavyweight libraries are NOT required for this
# demo/proof-of-concept stage.
#
# ============================================================
