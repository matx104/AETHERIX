# Triage Labels

| Label | Meaning |
| ----- | ------- |
| `needs-triage`     | Maintainer to evaluate |
| `needs-info`       | Waiting on reporter |
| `ready-for-agent`  | AFK-ready |
| `ready-for-human`  | Needs human |
| `wontfix`          | No-go |

## Research-flavour labels

- `module:routing`, `module:security`, `module:orbital`, `module:simulation`, `module:infrastructure` — scope to one src module
- `qkd` — Quantum Key Distribution work
- `rl-policy` — RL routing model / training
- `bpv7` — Bundle Protocol v7 layer
- `demo` — affects `demos/` or `run_demos.sh`
- `paper` — paper/presentation/interview material
- `reproducibility` — issue blocks reproducing a documented result
